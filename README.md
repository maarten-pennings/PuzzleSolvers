# Puzzle solvers

I sometime stumble on a puzzle that intrigues me enough to write a solver for it.
Here is my small collection.


## Futoshiki (2026-01)

Complete a latin square with some extra greater-than constraints.

[Solver](futoshiki)


## Mesh (2023-12)

A puzzle with interlocking bars, a bit like the # symbol.

[Solver](meshpuzzle)


## Digits (2020-12)

Fit tiles in the form of 7-segments digits in a rectangle.

[Solver](digitspuzzle)


## Solve 1567 (2020-04)

Using only the four numbers 1, 5, 6, 7 form an expression with just +, -, *, / that yields 21.

[Solver](solve1567)


## Calcudoku (2016-12)

Calcudoku is a sort of crossword puzzle with numbers.

[Solver](calcudoku)


## Knight's tour (2013-07)

Can a knight make a tour on a chess board?

[Solver](knight)


## Eight queens (2012-07)
